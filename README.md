# Using

    $ go get github.com/tarsisazevedo/benchstats
    $ benchstats <url> <times to access>

    DNS Lookup: 0.002307574s
    TCP Connections: 0.139763736s
    Server Procesing: 0.152835476s
    Server Tranfer: 4.2688e-05s
